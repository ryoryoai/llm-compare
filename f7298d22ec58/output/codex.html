    知財・法務・開発それぞれ忙しく、継続的な探索が止まる。
  • 情報が散らばり、判断材料にならない
    候補はあっても、根拠整理（対応関係）や優先順位が作れない。

  • ROIが見えず、意思決定が遅れる
    売上規模（推定）が分からず、どこから着手すべきか決めにくい。

選ばれる理由（USP）

調査の「網羅性」と「価値ある特許の見極め」を両立し、次のアクションに繋がるアウトプットへ。

技術者 × 知財専門家の高度分析

技術的整合性と知財観点の妥当性を両立。単なる検索結果ではなく、判断材料として使える形に整理します。

AI活用による網羅的グローバル調査

複数の高性能AIツールと独自プロンプト等で探索範囲を拡張。偏り/取りこぼしを抑え、候補を広く拾います。

自らライセンシングを行う主体の視点

「見つける」だけではなく、収益化/交渉に繋がる価値ある特許を見極める視点で、優先度付けに耐える材料を作ります。

ICP（理想顧客）

特許はある。だが、侵害発見の体制が足りない。

特許を数十件以上保有し、侵害発見に割けるリソースが不足している企業/組織。 かつ、公表情報から侵害の有無を判断しやすい領域（ソフトウェア/ICT/機械/電気電子/日用品など）で成立しやすい設計です。

対象技術分野（得意領域 / 原則対象外）

公表情報（外観・UI・仕様・公開資料・Web情報等）から侵害の有無を判断しやすい分野に強みがあります。 一方、分解や成分分析等が前提となる分野は原則対象外です。

得意：公表情報から判断しやすい分野

外観/画面/UI/仕様/提供形態など、公開情報で「要件対応」を検討しやすい領域で成果が出やすい設計です。

  • ソフトウェア / ICT / IoT / AI / ビジネスモデル（UI・機能・運用フロー等）
  • デバイス / 機械 / 電気電子 / 制御（構造・仕様・公開資料等）
  • 構造物 / 日用品（外観・構成・公開情報等）

原則対象外：分解・分析が必要な分野

公表情報だけでは判断が難しく、分解や成分分析等が必要となるため、本サービスの前提に合いません。

注意事項（非保証 / 推定 / 範囲）

本サービスは「侵害の断定」ではなく、収益化/交渉の入口となる初動情報を整えるための調査・整理です。 透明性のため、前提と範囲を明示します。

非保証・推定に関する重要事項

  • 侵害可能性は「可能性」の整理であり、侵害の断定/保証ではありません。
  • 累計売上は推定値を含み、入手可能な公開情報等に基づき算出します。
  • 公表情報では追えない要件がある場合、追加調査や別手段が必要になることがあります。
  • 対象外領域/要件に該当する場合、調査成立が難しい可能性があります（事前確認します）。

本サービスの範囲（できる / できない）

  • できる：被疑侵害製品の探索、候補の整理、簡易チャート、売上推定、優先度の材料作成
  • 範囲外：鑑定書の作成、侵害訴訟対応、交渉代理（必要に応じて別途専門家対応）
  • 目的：「次の検討」に進むための初動情報を整える
  • 使いどころ：社内稟議・優先度付け・追加調査の判断・外部専門家への引き継ぎ
「見える化」した上で、次の意思決定へ
本サービスは、断定ではなく「候補と根拠の整理」を目的とします。必要に応じて、詳細評価（専門家）へ繋げられる形式で整えます。

FAQ

よくある質問に、短く・具体的にお答えします（最低限の前提も明示します）。

どの程度の情報があれば相談できますか？
特許の番号/公報リストがあれば相談可能です。対象領域と「何を判断したいか」（優先度付け、候補抽出、社内説明材料など）も併せて共有いただくと、調査設計と見積がスムーズです。
成果物は、社内稟議や外部専門家への引き継ぎに使えますか？
はい。候補製品/企業、簡易クレームチャート、累計売上（推定）など、検討に必要な材料をセットで整理します。侵害の断定ではありませんが、「次の検討」に進むための土台になります。
AIだけで自動判定するサービスですか？
いいえ。AIは網羅性を補強するために活用しますが、技術者と知財専門家（弁理士等）が関与し、根拠整理と妥当性確認を行います。特許は一件ずつ調査します。
売上の調査はどこまで正確ですか？
推定値を含みます。公開情報等に基づき算出するため、完全な正確性を保証するものではありません。意思決定のための「目安」として、優先度付けに活用いただく設計です。
調査できない分野はありますか？
原則として、製薬・化学・材料・半導体等のように分解や成分分析が必要な分野は対象外です。公表情報で成立するか個別に確認しますので、まずはご相談ください。
侵害の「断定」や鑑定書の作成もお願いできますか？
本サービスは初動情報の可視化が目的で、侵害の断定/保証や鑑定書作成、交渉代理は範囲外です。必要に応じて、別途専門家による詳細評価へ繋げやすい形式で整理します。
どのくらいの件数から依頼できますか？
まずは小さく開始して成立条件とアウトプット粒度を合わせる進め方も可能です。価格は「特許100件あたり 30万〜50万円（税別）」が目安で、件数/難易度で個別見積となります。
秘密保持や社内規程の都合があるのですが大丈夫ですか？
まずは公報番号等の公開情報ベースでご相談可能です。共有範囲や取り扱い条件は、貴社の規程に合わせて調整します。
UI/ソフトウェアの特許でも成立しますか？
はい。特にUI・機能・運用フローなどが公開情報で追える場合、成立しやすい領域です。対象国/市場や情報入手性も含めて、初回相談で成立条件を確認します。
すでに候補製品があるのですが、それでも価値はありますか？
はい。候補がある場合でも、根拠整理（簡易チャート）、追加候補の探索、売上推定による優先度付けで、意思決定の精度を上げやすくなります。
